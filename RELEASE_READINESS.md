Status: **Not public-release ready**

This project is usable as a local study app, but it is not certified for public release or distribution as fully factual/rights-cleared content.

## Current Allowed Use

- Local personal study.
- Public-domain primary-text exploration where source references are recorded.
- Draft review of lecture-derived interpretation with visible caution labels.

## Not Allowed Without Further Clearance

- Claiming the whole project is fully fact-certified.
- Publishing lecture-derived pages as rights-cleared source material.
- Presenting broad lecture theses as neutral historical fact.
- Reusing modern copyrighted article/book text beyond short references without permission.
