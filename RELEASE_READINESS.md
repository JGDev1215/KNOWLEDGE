# Release Readiness

Status: **Full app not public-release ready**

This project is usable as a local study app, but it is not certified for public release or distribution as fully factual/rights-cleared content.

## Public-Domain-Only Release Build

The project now has a public-domain-only release path:

```sh
npm run audit:public
```

That command builds with `VITE_CONTENT_SCOPE=public-domain`, swaps in public-only content/provenance modules, and verifies that the generated bundle excludes lecture-derived, mixed-source, unverified, and modern-author material. This mode currently includes only the public-domain primary-text works:

- Dante, `Divine Comedy`, Longfellow translation
- Homer, `Iliad`, Samuel Butler translation
- Milton, `Paradise Lost`

This public-domain subset is the only mode that can be treated as release-cleared from the current repository evidence.
After the public build finishes, the scripts restore the full local content/provenance modules for normal development.
Full and public builds should be run sequentially because they use generated content/provenance modules.

## Blocking Items

| Blocker | Reason | Required clearance |
| --- | --- | --- |
| Lecture transcript rights | Seven tracked raw scripts are lecture-derived and ownership/permission is not documented. | Confirm that the project owner has rights to publish the transcripts and derived study pages, or remove/private-scope them. |
| Modern author material | Gay Talese-related content concerns a living author and copyrighted article/book material. | Keep only short factual references unless permission exists; avoid extended copyrighted text. |
| Missing raw provenance | `great_books_1_secrets_of_the_universe.html` and `great_books_5_the_odyssey.html` lack matching tracked raw scripts. | Add source scripts/edition metadata or mark as draft-only. |
| Broad lecture claims | Several lecture-derived pages contain strong causation or literary-history claims. | Reword as lecture interpretation, remove, or cite scholarly sources. |

## Current Allowed Use

- Local personal study.
- Public-domain-only release build generated by `npm run audit:public`.
- Public-domain primary-text exploration where source references are recorded.
- Draft review of lecture-derived interpretation with visible caution labels.

## Not Allowed Without Further Clearance

- Claiming the whole project is fully fact-certified.
- Publishing lecture-derived pages as rights-cleared source material.
- Presenting broad lecture theses as neutral historical fact.
- Reusing modern copyrighted article/book text beyond short references without permission.
