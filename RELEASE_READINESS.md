# Release Readiness

Status: **Release-cleared for the tracked public source set**

This repository now tracks only the public-domain primary-text app source. Lecture-derived pages, raw lecture transcripts, missing-provenance pages, and modern-author lecture material were removed from tracked public source.

## Release Build

```sh
npm run release:verify
```

That command builds with `VITE_CONTENT_SCOPE=public-domain`, swaps in public-only content/provenance modules, and verifies that the generated bundle excludes removed lecture-derived, mixed-source, unverified, and modern-author material. It delegates to `npm run audit:public`, which performs both static bundle checks and public-mode browser workflow checks.

The release-cleared tracked works are:

- Dante, `Divine Comedy`, Longfellow translation
- Homer, `Iliad`, Samuel Butler translation
- Milton, `Paradise Lost`

`npm run build` uses the same public-domain-only bundle path. `npm run build:local` currently builds the same cleared set because no private/local-only source files remain tracked.

## Current Allowed Use

- Public-domain primary-text exploration where source references are recorded.
- Public release build generated by `npm run release:verify`.
- Default production build generated by `npm run build`.

## Not Allowed Without Further Evidence

- Reintroducing lecture transcripts or transcript-derived pages without written permission, ownership evidence, license grant, or public-domain status.
- Reintroducing missing-provenance pages without raw source origin, edition metadata, and rights/provenance record.
- Reusing modern copyrighted article/book text beyond short factual references without permission.

`RIGHTS_CLEARANCE.md` records the removal decision for prior local-only material.
